codex-graph
==============

Highly customizable light weight graph plugin







==============

## Basic usage

initialize the plugin 

$("body").codexgraph({});

## Settings



    $("body").codexgraph({
    knobcolor: '#14b9d5',
	fillcolor: '#fff',
	textcolor: '#000',
    });
